# Task 3 | i-Tree Eco Modelling

The i-Tree Eco model is a tool for estimating the ecosystem services of trees in urban areas. The model is developed by the USDA Forest Service and is freely available for download at [itreetools.org](https://www.itreetools.org/tools/i-tree-eco).
